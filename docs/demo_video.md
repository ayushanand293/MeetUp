# Demo Video

Use this as the shot list for a 60-90 second interviewer demo.

## Shot List

- `0:00-0:10`: Show the app opening on two simulators side-by-side.
- `0:10-0:25`: Show phone OTP sign-in using demo-safe accounts.
- `0:25-0:40`: Show Find Friends, then send a meet request or invite.
- `0:40-0:55`: Accept on the second device and enter one active 1:1 session.
- `0:55-1:10`: Show realtime peer movement and session freshness.
- `1:10-1:25`: Show optional meet-at-place destination, distance/open-in-maps, and routing if `EXPO_PUBLIC_ORS_KEY` is set.
- `1:25-1:30`: End the session and show both devices return to a safe state.

Do not show real phone numbers, real locations, private contacts, credentials, JWTs, API keys, or production data.

## Recording Setup

- Preferred: two iOS simulators side-by-side on Mac.
- Alternative: one simulator plus one physical test device with demo-safe data.
- Record with QuickTime Player: `File > New Screen Recording`.
- Keep the video tight; the goal is to prove the flow, not explain every screen.
